# skilltree-app

## Tech Stack

### Front End

-   Vue 3
-   Pinia
-   Vue Router
-   Vite
-   D3.js
-   Pixi.js

### Back End

-   Node with Express
-   MariaDB

## Project Setup

### Install dependancies

npm install

### Setup database

Please import the latest DB backup from the "db_backups" folder.

## Run or Compile

### Hot-Reload for Development

npm run dev

### Build

npm run build

### Run compiled version

npm run start
